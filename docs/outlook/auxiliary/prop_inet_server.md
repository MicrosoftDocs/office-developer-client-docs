# PROP_INET_SERVER

Represents the server name of a general Internet mailbox.

|Key |Value |
|:-----|:-----|
|Identifier:  <br/> |0x0100  <br/> |
|Property type:  <br/> |PT_UNICODE  <br/> |
|Property tag:  <br/> |0x0100001F  <br/> |
|Access:  <br/> |Read-only  <br/> |
