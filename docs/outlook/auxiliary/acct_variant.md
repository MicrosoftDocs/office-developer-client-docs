# ACCT_VARIANT

A variable of this data type holds the value of a property, which is of a variant data type.
  
## Quick info

```
typedef struct 
    { 
        DWORD dwType; 
        union  
            { 
            DWORD dw; 
            WCHAR *pwsz; 
            ACCT_BIN bin; 
            } Val; 
    } ACCT_VARIANT; 

```

## Members

 _dwType_
  
> Type of variant:
    
    - PT_LONG
    
    - PT_UNICODE
    
    - PT_BINARY
    
 _dw_
  
> DWORD value of variant.
    
 _pwsz_
  
> String value of variant.
    
 _bin_
  
> Binary value of the variant.
