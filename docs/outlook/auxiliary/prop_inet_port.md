# PROP_INET_PORT

Represents the port number for a general Internet mailbox.
  
## Quick Info

|||
|:-----|:-----|
|Identifier:  <br/> |0x0104  <br/> |
|Property type:  <br/> |PT_DWORD  <br/> |
|Property tag:  <br/> |0x01040003  <br/> |
|Access:  <br/> |Read-only  <br/> |
   
## Remarks

An example is 110 for POP accounts.
