# IEnumFBBlock::Reset

Resets the enumerator by setting the cursor to the beginning.
  
## Quick Info

See [IEnumFBBlock](ienumfbblock.md).
  
```
HRESULT Reset();
```

## Return Values

S_OK if the call succeeded; otherwise, an error code.
