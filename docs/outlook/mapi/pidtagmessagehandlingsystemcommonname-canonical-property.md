# PidTagMessageHandlingSystemCommonName Canonical Property

  
  
**Applies to**: Outlook 
  
Contains the common name of the message handling system.
  
|||
|:-----|:-----|
|Associated properties:  <br/> |PR_MHS_COMMON_NAME, PR_MHS_COMMON_NAME_A, PR_MHS_COMMON_NAME_W  <br/> |
|Identifier:  <br/> |0x3A0F  <br/> |
|Data type:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Area:  <br/> |Address  <br/> |
   
## Remarks

These properties contain the common name of the message handling system.
