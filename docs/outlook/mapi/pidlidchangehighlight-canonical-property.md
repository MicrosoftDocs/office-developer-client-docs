# PidLidChangeHighlight Canonical Property

  
  
**Applies to**: Outlook 
  
Specifies a bit field that indicates how the meeting object changed.
  
|||
|:-----|:-----|
|Associated properties:  <br/> |dispidChangeHighlight  <br/> |
|Property set:  <br/> |PSETID_Appointment  <br/> |
|Long ID (LID):  <br/> |0x00008204  <br/> |
|Data type:  <br/> |PT_LONG  <br/> |
|Area:  <br/> |Meetings  <br/> |
   
## Remarks

This property is not required. The individual flags that can be set are detailed in [[MS-OXOCAL]](http://msdn.microsoft.com/library/09861fde-c8e4-4028-9346-e7c214cfdba1%28Office.15%29.aspx).
