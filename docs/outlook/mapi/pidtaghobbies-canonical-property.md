# PidTagHobbies Canonical Property

  
  
**Applies to**: Outlook 
  
Contains the names of the hobbies of the messaging user.
  
|||
|:-----|:-----|
|Associated properties:  <br/> |PR_HOBBIES, PR_HOBBIES_A, PR_HOBBIES_W  <br/> |
|Identifier:  <br/> |0x3A43  <br/> |
|Data type:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Area:  <br/> |MAPI mail user  <br/> |
   
## Remarks

These properties provide identification and access information about a messaging user and the content is defined by the user's organization.
