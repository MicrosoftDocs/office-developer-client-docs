# PidTagPostalAddress Canonical Property

  
  
**Applies to**: Outlook 
  
Contains the recipient's postal address.
  
|||
|:-----|:-----|
|Associated properties:  <br/> |PR_POSTAL_ADDRESS, PR_POSTAL_ADDRESS_A, PR_POSTAL_ADDRESS_W  <br/> |
|Identifier:  <br/> |0x3A15  <br/> |
|Data type:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Area:  <br/> |MAPI mail user  <br/> |
   
## Remarks

These properties provide identification and access information for a recipient. They are defined by the recipient and the recipient's organization.
