# PidTagHomeFaxNumber Canonical Property

  
  
**Applies to**: Outlook 
  
Contains the telephone number of the recipient's home fax machine. 
  
|||
|:-----|:-----|
|Associated properties:  <br/> |PR_HOME_FAX_NUMBER, PR_HOME_FAX_NUMBER_A, PR_HOME_FAX_NUMBER_W  <br/> |
|Identifier:  <br/> |0x3A25  <br/> |
|Data type:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Area:  <br/> |MAPI mail user  <br/> |
   
## Remarks

These properties provide identification and access information about a recipient. They are defined by the recipient and their organization.
