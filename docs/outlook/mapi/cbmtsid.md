# CbMTSID

 **Last modified:** March 09, 2015 
  
 * **Applies to:** Outlook * 
  
Computes the number of bytes in an existing [MTSID](mtsid.md) structure. 
  
|||
|:-----|:-----|
|Header file:  <br/> |Mapidefs.h  <br/> |
|Related structure:  <br/> |**MTSID** <br/> |
   
```

```

## Parameters

 __lpentry_
  
> Pointer to the existing **MTSID** structure.
