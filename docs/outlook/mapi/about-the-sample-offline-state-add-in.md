# About the Sample Offline State Add-in

  
  
**Applies to**: Outlook 2013 | Outlook 2016 
  
The Offline State API supports callbacks indicating changes in a user's connection state in Outlookâ€”for example, from being online in Outlook to being offline. The Sample Offline State Add-in is a COM add-in written in C++ that demonstrates how to receive notifications of connection state changes and how to modify the current state using the Offline State API. For more information about the Offline State API, see [About the Offline State API](about-the-offline-state-api.md).
