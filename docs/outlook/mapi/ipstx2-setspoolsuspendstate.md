# IPSTX2::SetSpoolSuspendState

  
  
**Applies to**: Outlook 
  
Sets the suspended state on the spooler.
  
```cpp
void SetSpoolSuspendState( 
    ULONG ulState 
);
```

## Parameters

 _ulState_
  
> [in] The state to set the spooler to. It must be one of the following values:
    
 **SS_ACTIVE**
  
> 
    
 **SS_SUSPENDED**
  
>
