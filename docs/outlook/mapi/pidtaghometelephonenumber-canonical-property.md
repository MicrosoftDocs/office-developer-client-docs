# PidTagHomeTelephoneNumber Canonical Property

 **Last modified:** March 09, 2015 
  
 * **Applies to:** Outlook * 
  
Contains the primary telephone number of the recipient's home. 
  
|||
|:-----|:-----|
|Associated properties:  <br/> |PR_HOME_TELEPHONE_NUMBER, PR_HOME_TELEPHONE_NUMBER_A, PR_HOME_TELEPHONE_NUMBER_W  <br/> |
|Identifier:  <br/> |0x3A09  <br/> |
|Data type:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Area:  <br/> |MAPI mail user  <br/> |
   
## Remarks

These properties provide identification and access for a recipient. They are defined by the recipient and their organization.
