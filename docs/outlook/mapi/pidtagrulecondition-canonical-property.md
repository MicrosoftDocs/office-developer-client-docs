# PidTagRuleCondition Canonical Property

  
  
**Applies to**: Outlook 2013 | Outlook 2016 
  
The condition used when you evaluate the rule.
  
|||
|:-----|:-----|
|Associated properties:  <br/> |PR_RULE_CONDITION  <br/> |
|Identifier:  <br/> |0x6679  <br/> |
|Data type:  <br/> |PT_SRESTRICTION  <br/> |
|Area:  <br/> |Server Side Rules  <br/> |
   
## Remarks

The condition is expressed as a **Restriction** and the **PropertyValue** buffer contains the **Restriction** structure packaged as specified in [[MS-OXCDATA]](https://msdn.microsoft.com/library/1afa0cd9-b1a0-4520-b623-bf15030af5d8%28Office.15%29.aspx).
