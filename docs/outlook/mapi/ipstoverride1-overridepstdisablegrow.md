# IPSTOVERRIDE1::OverridePSTDisableGrow

  
  
**Applies to**: Outlook 
  
Unlocks a Personal Folders (.pst) file for a session, overriding the PSTDisableGrow policy.
  
```cpp
HRESULT OverridePSTDisableGrow();
```

## Return value

S_OK
  
> The function call was successful.
