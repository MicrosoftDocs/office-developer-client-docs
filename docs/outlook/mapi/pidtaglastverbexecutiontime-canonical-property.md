# PidTagLastVerbExecutionTime Canonical Property

  
  
**Applies to**: Outlook 
  
Contains the time when the last verb was executed.
  
|||
|:-----|:-----|
|Associated properties:  <br/> |PR_LAST_VERB_EXECUTION_TIME  <br/> |
|Identifier:  <br/> |0x1082  <br/> |
|Data type:  <br/> |PT_SYSTIME  <br/> |
|Area:  <br/> |History  <br/> |
