# PidNameLineCount Canonical Property

  
  
**Applies to**: Outlook 2013 | Outlook 2016 
  
Specifies the number of lines in the file that is attached to the document.
  
|Property |Value |
|:-----|:-----|
|Friendly names:  <br/> |None  <br/> |
|Property set:  <br/> |PS_PUBLIC_STRINGS  <br/> |
|Property name:  <br/> |LineCount  <br/> |
|Data type:  <br/> |PT_LONG  <br/> |
|Area:  <br/> |Common  <br/> |
