# PidTagLastVerbExecuted Canonical Property

  
  
**Applies to**: Outlook 
  
Contains the last verb executed.
  
|||
|:-----|:-----|
|Associated properties:  <br/> |PR_LAST_VERB_EXECUTED  <br/> |
|Identifier:  <br/> |0x1081  <br/> |
|Data type:  <br/> |PT_LONG  <br/> |
|Area:  <br/> |History  <br/> |
