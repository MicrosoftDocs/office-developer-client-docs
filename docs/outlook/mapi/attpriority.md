# attPriority

  
  
**Applies to**: Outlook 
  
MAPI message priorities are mapped to TNEF values for backward compatibility as follows:
  
|**Priority**|**MAPI value**|**TNEF value**|
|:-----|:-----|:-----|
|normal  <br/> |-1  <br/> |3  <br/> |
|high  <br/> |0  <br/> |2  <br/> |
|low  <br/> |1  <br/> |1  <br/> |
