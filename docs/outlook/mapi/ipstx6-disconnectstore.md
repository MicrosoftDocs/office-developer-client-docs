# IPSTX6::DisconnectStore

  
  
**Applies to**: Outlook 2013 | Outlook 2016 
  
Disconnects the local store.
  
```cpp
HRESULT DisconnectStore( 
    ULONG ulFlags 
);
```

## Parameters

 _ulFlags_
  
> [in] Flags to modify behavior. This parameter must always be set to 0.
