# CbADRLIST

  
  
**Applies to**: Outlook 
  
Computes the number of bytes in an existing [ADRLIST](adrlist.md) structure. 
  
|||
|:-----|:-----|
|Header file:  <br/> |Mapidefs.h  <br/> |
|Related structure  <br/> |**ADRLIST** <br/> |
   
```
CbADRLIST (_lpadrlist)
```

## Parameters

 __lpadrlist_
  
> Pointer to the existing **ADRLIST** structure.
