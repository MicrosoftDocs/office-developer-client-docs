# PidNameLastAuthor Canonical Property

  
  
**Applies to**: Outlook 2013 | Outlook 2016 
  
Specifies the most recent author of the attached file.
  
|Property|Value|
|:-----|:-----|
|Friendly names:  <br/> |None  <br/> |
|Property set:  <br/> |PS_PUBLIC_STRINGS  <br/> |
|Property name:  <br/> |LastAuthor  <br/> |
|Data type:  <br/> |PT_UNICODE  <br/> |
|Area:  <br/> |Common  <br/> |
