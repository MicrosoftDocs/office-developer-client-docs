# PidTagCompanyName Canonical Property

  
  
**Applies to**: Outlook 
  
Contains the recipient's company name. 
  
|||
|:-----|:-----|
|Associated properties:  <br/> |PR_COMPANY_NAME, PR_COMPANY_NAME_A, PR_COMPANY_NAME_W  <br/> |
|Identifier:  <br/> |0x3A16  <br/> |
|Data type:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Area:  <br/> |Contact  <br/> |
   
## Remarks

These properties provide identification and access information about a recipient. They are defined by the recipient and the recipient's organization.
