# PidTagCountry Canonical Property

  
  
**Applies to**: Outlook 
  
Contains the name of the recipient's country/region. 
  
|||
|:-----|:-----|
|Associated properties:  <br/> |PR_COUNTRY, PR_COUNTRY_A, PR_COUNTRY_W, PR_BUSINESS_ADDRESS_COUNTRY, PR_BUSINESS_ADDRESS_COUNTRY_A, PR_BUSINESS_ADDRESS_COUNTRY_W  <br/> |
|Identifier:  <br/> |0x3A26  <br/> |
|Data type:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Area:  <br/> |Contact  <br/> |
   
## Remarks

These properties are examples of the properties that provides identification and access information about a recipient. These properties are defined by the recipient and the recipient's organization.
