# PidNameCompany Canonical Property

  
  
**Applies to**: Outlook 
  
Specifies the company for which the file was created.
  
|||
|:-----|:-----|
|Friendly names:  <br/> |None  <br/> |
|Property set:  <br/> |PS_PUBLIC_STRINGS  <br/> |
|Property name:  <br/> |Company  <br/> |
|Data type:  <br/> |PT_UNICODE  <br/> |
|Area:  <br/> |Common  <br/> |
