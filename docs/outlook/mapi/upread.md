# UPREAD

  
  
**Applies to**: Outlook 
  
Information for uploading the read state of items during the [upload read status state](upload-read-status-state.md).
  
## Quick Info

```
struct UPREAD 
{ 
    PUPREADE pupre; 
    UINT cEnt; 
};
```

## Members

 _pupre_
  
>  [out] Vector of **[UPREADE](upreade.md)** entries. 
    
 _cEnt_
  
>  [out] Number of **UPREADE** entries.
