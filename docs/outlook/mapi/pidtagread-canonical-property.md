# PidTagRead Canonical Property

  
  
**Applies to**: Outlook 
  
Indicates whether a table row has been read.
  
|||
|:-----|:-----|
|Associated properties:  <br/> |PR_READ  <br/> |
|Identifier:  <br/> |0x0E69  <br/> |
|Data type:  <br/> |PT_BOOLEAN  <br/> |
|Area:  <br/> |MAPI non-transmittable  <br/> |
