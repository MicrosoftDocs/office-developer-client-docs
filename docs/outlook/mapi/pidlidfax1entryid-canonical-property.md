# PidLidFax1EntryId Canonical Property

  
  
**Applies to**: Outlook 
  
Specifies the EntryID of the contact's business fax address.

|Associated properties:  <br/> |dispidFax1EntryID  <br/> |

|Data type:  <br/> |PT_BINARY  <br/> |
|Area:  <br/> |Contact  <br/> |
