# SBinary

  
  
**Applies to**: Outlook 
  
Describes a property of type PT_BINARY.
  
|||
|:-----|:-----|
|Header file:  <br/> |Mapidefs.h  <br/> |
   
```
typedef struct _SBinary
{
  ULONG      cb;
  LPBYTE     lpb;
} SBinary, FAR *LPSBinary;

```

## Members

 **cb**
  
> Count of bytes in the **lpb** member. 
    
 **lpb**
  
> Pointer to the PT_BINARY property value.
