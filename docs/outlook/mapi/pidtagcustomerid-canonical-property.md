# PidTagCustomerId Canonical Property

  
  
**Applies to**: Outlook 
  
Contains the contact's customer ID number.
  
|||
|:-----|:-----|
|Associated properties:  <br/> |PR_CUSTOMER_ID, PR_CUSTOMER_ID_A, PR_CUSTOMER_ID_W  <br/> |
|Identifier:  <br/> |0x3A4A  <br/> |
|Data type:  <br/> |PT_STRING8, PT_UNICODE  <br/> |
|Area:  <br/> |Contact  <br/> |
   
## Remarks

The contents of these properties are defined by the user or the user's organization.
