# PidTagRuleName Canonical Property

  
  
**Applies to**: Outlook 
  
Specifies the user-readable name of the rule.
  
|||
|:-----|:-----|
|Associated properties:  <br/> |PR_RULE_NAME  <br/> |
|Identifier:  <br/> |0x6682  <br/> |
|Data type:  <br/> |PT_STRING8  <br/> |
|Area:  <br/> |Server Side Rules  <br/> |
