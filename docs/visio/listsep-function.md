# LISTSEP Function

Returns the list-separator string for the current user locale.
  
## Syntax

LISTSEP ()
  
### Return value

String
  
## Example

SETF(GETREF(user.extent), "MAX(Width" &amp; ListSep() &amp; "Height)")
