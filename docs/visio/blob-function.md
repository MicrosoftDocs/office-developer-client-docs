# BLOB Function

Decodes a base64 string and stores it as binary data. 
  
## Syntax

BLOB (** *bintext* ** ) 
  
### Parameters

|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _bintext_ <br/> |Required  <br/> |**String** <br/> | A base64 encoded string.  <br/> |
