# IS1D Function

Returns TRUE if the shape is 1-D (one-dimensional); returns FALSE if the shape is 2-D (two-dimensional).
  
## Version Information

Version Added: Visio 2010 
  
## Syntax

IS1D()
  
### Return value

 **Boolean**
  
## Remarks

If the sheet (shape) is a page or a document, the IS1D function returns #REF!.
