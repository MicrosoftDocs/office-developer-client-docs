# ABS Function (VisioShapeSheet)

Returns the absolute value of a number.
  
## Syntax

ABS( *number*) 
  
### Parameters

|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Number_ <br/> |Required  <br/> |**Decimal** <br/> |The number whose absolute value you want to find. |
   
## Example

ABS(-1.24) 
  
Returns 1.24.
