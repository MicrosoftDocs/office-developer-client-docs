# PNTX Function

Returns the _x_-coordinate of a point.
  
## Syntax

PNTX(**_point_** )
  
### Parameters

|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _point_ <br/> |Required  <br/> |**Number** <br/> |The _x_-coordinate of the point. |

### Return value

Number
  
## Example

PNTX(PNT(7,12))
  
Returns 7.
