# LISTORDER Function

Returns the 1-based position of the shape in the list.
  
## Version Information

Version Added: Visio 2010 
  
## Syntax

LISTORDER()
  
### Return value

Integer
  
## Remarks

If the shape is not a list member, the LISTORDER function returns -1.
