# RAD Function

Converts the value of an angle from degrees to radians.
  
## Syntax

RAD(** *angle* ** ) 
  
### Parameters

|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _angle_ <br/> |Required  <br/> |**Number** <br/> |The angle to convert.  <br/> |
   
## Example

RAD(45) 
  
Returns 0.7854.
