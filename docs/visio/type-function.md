# TYPE Function

Returns an object's internal type as a number. 
  
## Syntax

TYPE ()
  
### Return value

Number
  
## Remarks

The following table lists internal object types as numbers.
  
|**Object**|**Number**|
|:-----|:-----|
|Page  <br/> |70  <br/> |
|Group  <br/> |71  <br/> |
|Shape  <br/> |72  <br/> |
|Style  <br/> |74  <br/> |
|Guide  <br/> |77  <br/> |
|Foreign  <br/> |78  <br/> |
|Document  <br/> |79  <br/> |
