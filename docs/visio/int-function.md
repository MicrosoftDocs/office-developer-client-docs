# INT Function

Rounds a number down to the next integer.
  
## Syntax

INT(** *number* ** ) 
  
### Parameters

|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _number_ <br/> |Required  <br/> |**Number** <br/> |The number to round down.  <br/> |
   
## Example 1

INT(7.2)
  
Returns 7.
  
## Example 2

INT(-7.2)
  
Returns -8.
