# PI Function

Returns the mathematical constant  *pi*  , which is approximately 3.1415926535898. 
  
## Syntax

PI( )
  
### Return Value

Number
  
## Example

(14 in / 2)^2 \* PI( ) 
  
Returns 153.9380 inches, the area of a circle whose diameter is 14 inches.
