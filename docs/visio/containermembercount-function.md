# CONTAINERMEMBERCOUNT Function

Returns the total number of shapes in the container.
  
## Version Information

Version Added: Visio 2010 
  
## Syntax

CONTAINERMEMBERCOUNT()
  
### Return value

 **Integer**
  
## Remarks

If the shape is not a container, CONTAINERMEMBERCOUNT returns -1.
