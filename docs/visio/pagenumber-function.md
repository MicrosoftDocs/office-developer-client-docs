# PAGENUMBER Function

Returns the number of the page that contains the shape. 
  
## Syntax

PAGENUMBER()
  
### Return value

Number
  
## Remarks

If the current page is a background page (unnumbered), or if an error occurs, the PAGENUMBER function returns zero.
