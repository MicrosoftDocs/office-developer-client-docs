# PNTY Function

Returns the  _y_-coordinate of a point.
  
## Syntax

PNTX(** *point* ** ) 
  
### Parameters

|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _point_ <br/> |Required  <br/> |**Number** <br/> |The  _y_-coordinate of the point.  <br/> |
   
### Return value

Number
  
## Example

PNTY(PNT(7,12)) 
  
Returns 12.
