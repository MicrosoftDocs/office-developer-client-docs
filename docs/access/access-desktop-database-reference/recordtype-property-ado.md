# RecordType Property (ADO)


**Applies to**: Access 2013 | Office 2013

Indicates the type of [Record](record-object-ado.md) object.

## Return Value

Returns a [RecordTypeEnum](recordtypeenum.md) value.

## Remarks

The **RecordType** property is read-only.
